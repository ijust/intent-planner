# 外部調査の記録

内部に根拠がない専門的な観点を、利用者の明示承認を得て外部調査したときの記録置き場です。調査ごとに `.intent/research/<日付>-<話題>/` を作ります。既存の調査記録をこの形に一括変更する必要はありません。

## `README.md` に残す内容

- 調査の目的と、閉じたい判断
- 実行日と利用した方法
- 利用者が承認した検索文または調査依頼文
- 外部へ送った情報の範囲と承認
- 調査IDと現在の状態（手段が返す場合）

正確な文面にリポジトリへ残せない情報が含まれるときは、一般化または伏字した文面と、別承認を得た事実だけを残します。

## `synthesis.md` に残す内容

- 出典に支えられた専門的な観点の候補
- 利用者への質問候補
- 利用者の回答や提供資料との食い違い
- 根拠が得られず残った不明点
- 利用者が採用、非採用、または保留した内容
- 出典URL（一次情報を優先）

生の報告は任意です。通常の Intent Planning では `synthesis.md` を先に読み、必要な場合だけ生の報告を参照します。調査結果は候補であり、利用者が採用するまで Intent Tree、Intent Compass、packet へ自動転記しません。
