# External research records

This directory records external research that supplemented a specialist perspective missing from internal material after explicit user approval. Create `.intent/research/<date>-<topic>/` for each research run. Existing research records do not need to be bulk-migrated to this shape.

## What to keep in `README.md`

- The research purpose and the decision it was intended to close
- Execution date and method used
- The query or research request the user approved
- The outgoing-information boundary and its approval
- Research ID and current status when the method provides them

When the exact wording contains information unsafe to retain in the repository, keep a generalized or redacted version and only the fact that separate approval was obtained.

## What to keep in `synthesis.md`

- Source-backed specialist-perspective candidates
- Candidate questions for the user
- Contradictions with user answers or supplied material
- Unknowns left because reliable evidence was not found
- What the user adopted, declined, or deferred
- Source URLs, preferring primary sources

A raw report is optional. Normal Intent Planning reads `synthesis.md` first and opens raw material only when needed. Research results are candidates: never auto-write them into the Intent Tree, Intent Compass, or a packet before user adoption.
